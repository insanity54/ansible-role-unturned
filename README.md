# Ansible Unturned

Installs Unturned along with Rocketmod.

## Prerequisites

* Ansible >= 2.2
* Ubuntu 14.04 Server
* Steam account without Steam Guard disabled (for security, please create a NEW steam account for this purpose.)
* Rocketmod API key (https://rocketmod.net/api)
